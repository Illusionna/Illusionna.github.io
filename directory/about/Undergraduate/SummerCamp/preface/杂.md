# 关于夏令营申请

佬都在海投，你不海投？

# 关于套磁

同一个课题组最好只套一个老师，同一个院系下多套几个老师。我套老师的原则是：看面相、看最近的文章是否全是一作、看综合实力。建议夏令营活动正式开始半个月前联系。能要到导师微信，能提前交流面试，成功一大步。

# 关于推荐信

不少申请院校要求副高级（副教授）以上的推荐信 $1\sim2$ 封，如果和你的导师关系好，他们都是乐意帮忙的，如果关系一般，建议自己模仿导师口吻写好后，去办公室踢踢门，再让导师看看并签字。

# 关于材料盖章

很多夏令营报名提交的材料都需要有学院盖章，这是很令人烦的事情，所以可以把材料整理好，找辅导员一并盖章，或者能申请到电子章更好。我后期是自己写个 CV 代码把章给扣出来，然后 PS 把章盖到材料上，最后打印扫描。

# 关于个人陈述

一般申请院校都是 1000/2000/3000 字要求，甚至还有英文的，所以我觉得老师没时间看，就草草敷衍了事。

# 关于考核

绝大多数夏令营都有考核，其中面试考核是必须的，毕竟作为人与人之间的交流，笔试、机试考核根据院校课题组有可能会安排。此外，还有一些学校往年被鸽怕了，开夏令营费时费力费资金，所以就变成线上宣讲营，或者线下宣讲。

# 关于报销

一般夏令营会报销，且高铁二等座以下，有的是入营即可报销（像清北、中科院这种特别有钱的）、还有的是优秀营员给报销（这种很多）、还有最终研究生去了对方院校才给报销（这种也很多），当然也有不给报销的（但我只听说没见过）。

# 关于着装

大夏天的，正常穿搭就好了，短衣短裤、长衣长裤、衬衫连衣裙，别什么西装穿搭，热不死你，还有一群学生老师对你行注目礼。

# 关于住宿饮食

绝大多数院校免费提供。

# 关于效力

越临近最终填系统，效力越高，这是肯定的。很多学校在对营员完成考核后，成绩其实都已经定下来了，但院校不会公布，防止鸽子。有的院校效力是铁定的，有的院校效力是含糊其辞的，这都很正常，提前多了解。

# 关于一个学校申请多个专业方向

很多学校夏令营是一个系统，只允许申请一个方向，但也有不少可以申请多个方向。

# 关于英语

部分院校会要求英语自我介绍、英文文献翻译、英语对答。
